# Console
The Console Class Allows us to interact with the console/terminal in 5 main ways
   ```csharp
      Console.Print("Hello World");
      Console.Input("Enter a Number");
      Console.Clear();
      Console.Beep(x,y);
      Console.Exec("Console.Print("Hello world!");");
   ```
